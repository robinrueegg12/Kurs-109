Allgemeines über Cloud

Arten von Cloud

  • Public-Cloud (Kunden teilen sich eine Infrastruktur; für sensible Daten weniger geeignet)
  • Private-Cloud (Wird beim Anbieter zur Verfügung gestellt / im eigenen Haus betrieben)
  • hybried-Clod (Mischung aus Private und Public Cloud)

Cloud Computing

Einfach erklärt kann man mit Cloud-Computing die IT Infrastruktur mieten, anstatt sie zu kaufen. Anstatt in Datenbanken, Software und Hardware zu investieren, entscheiden sich Unternehmen dafür>

Cloud Services

  • Software as a Service (SaaS)
  • Platform as a Service (PaaS)
  • Infrastructure as a Service (IaaS)

AWS (Amazon Web Services)

Pay as you go

Das Prinzip "Pay-as-you-go" bei AWS Cloud ermöglicht es, nur für die tatsächlich genutzten Ressourcen und Services zu bezahlen, ohne langfristige Verpflichtungen oder Vorauszahlungen.
